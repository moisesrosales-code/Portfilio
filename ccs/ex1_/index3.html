Wild Wallpaper Cat

An interactive animated cat wallpaper built with p5.js, featuring five cats with eye-tracking and mouth animation.

I created an animated cat wallpaper with one main cat in the center and four smaller cats placed diagonally to form an X. Each cat has pointed ears, expressive eyes, detailed whiskers, and a mouth that moves. The eyes follow the mouse cursor, making the wallpaper interactive. The background flickers gently over time. Through this project I learned how to group shapes, make repeating patterns, and animate based on time and user interaction.
