Paint Program

A simple paint program with three brushes built using p5.js.

For this project, I designed a simple paint program with three brushes using p5.js. I am a beginner, and I wanted to get the hang of using functions, loops, and mouse interaction, so I did not overcomplicate it. The color palette, inspired by one of my own shirts, consists of dark charcoal, dusty rose, rust, beige, and blue. The three brushes are a simple paintbrush, a texture brush, and a structure brush.
